##Cloud Computing
Cloud computing is een concept dat een wereldwijde toegang biedt aan gedeelde IT-resources die worden beheerd door een cloud service provider en niet door het bedrijf dat gebruik maakt van de middelen.  
De voordelen om voor cloud computing te kiezen zijn:  
1. Availibility: Users hebben altijd de mogelijkheid om op hun services toegang te krijgen op eender welk moment.  
2. Collaboration: Users zien de cloud als een manier om tegelijkertijd te werken aan gemeenschappelijke data en bestanden.  
3. Elasticity: Aan de hand van de user zijn behoeften kunnen dynamisch zijn services aan gepast worden.  
4. Lage infrastructuur kost: Het concept van pay-per-use laat een organisatie toe om enkel te betalen wat ze verbruiken. Ze hebben geen onderhouod of upgrade kosten aan hun infrastructuur.  
5. Mobility: Users hebben toegang van overal op de wereld tot hun data en applicatie's.  
6. Minder risico: Organisaties kunnen nu eerst in de cloud ideën testen voor investeringen te maken in een technologie.  
7. Scalability: Users hebben toegang tot een groot aantal resources die scalen aan de hand van hun vragen.  
8. Virtualization: elke user kan alle beschikbare middelen zien. Daarom kan een provider een groter aantal users bedienen met minder fysieke middelen.  

Het doel van dit concept is de geautomatiseerde diensten voor provisioning makkelijk schaalbaar maken. Cloud computing heeft 3 servicemodellen, namelijk software as a service (SaaS), platform as a service (PaaS)
en infrastructuur as a service (IaaS). Gezien de groeiende belangstelling voor cloud computing is er een groei gebeurd in research over hoe software development beïnvloed word door cloud computing. Het assortiment van cloud service oplossingen is zeer groot. Samen met bedrijven die gespecialiseerd zijn in cloud diensten, hebben de IT reuzen zoals Microsoft, Amazon en Google ook hun eigen cloud service ontwikkeld.  
Azure wat de microsoft cloud service is biedt zowel IAAS en PAAS-diensten aan. In tegenstelling van zijn concurrenten, die enkel provisioning hardware hulpmiddelen hebben. Azure ondersteund ook het ontwikkelen van software applicaties. En tot tegenstelling met Google ondersteund Azure een hele reeks aan programmeertalen.
